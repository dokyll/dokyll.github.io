About Dokyll

About Dokyll

Dokyll è una CLI che trasforma i tuoi file .html,.md e tutti quelli del tuo sito in build statiche pronte per il web, con un flusso semplice e ripetibile, in formato .dok.

Scarica ora per Windows x64

Stai cercando per un altro sistema o architettura? Tutti i download

Perchè esiste Dokyll?

Dokyll nasce dalla curiosità: l'autore, avendo tante GitHub Pages, si disse: non voglio solo guardare la build, volgio vedere come si fa.
Così nasce Dokyll.

Build in file .dok

Scrivi il contenuto una sola volta, poi usa Dokyll per generare il sito statico in modo rapido e affidabile.

Automazione intelligente

La CLI gestisce template, asset e struttura, per concentrarti solo sui contenuti e sul risultato finale.

Output statico

Il risultato è un sito statico ottimizzato, facile da pubblicare su qualsiasi hosting e perfetto per performance e sicurezza.

Esempi di progetti

Usa Dokyll per generare pagine, blog o documentazione partendo dai tuoi file del sito.
