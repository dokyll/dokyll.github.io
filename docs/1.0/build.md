# Fare la build

Ora dobbiamo buildare il nostro blog.

Digitate nel terminale:

```bash
dokyll build-pages blog
```

e se tutto è andato bene, dovra dire:

```output
Congratulazioni! La tua build è disponibile in blog.dok!
```

Ora **Congratulazioni!** Sai creare un blog e buildare le pagine!
