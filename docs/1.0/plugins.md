# Usare i plugins

Nella 1.0.1 o successivo abbiamo introdotto i plugins. Installiamone uno!


Digitate nel terminale:

```bash
dokyll install-plugin test
```

dovrebbe apparire:

```text
[━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━] 100.0%
Plugin installato correttamente.
```

Poi digitate:

```bash
dokyll plugin test
```

dovrebbe apparire:


```text
Funziona!
```
