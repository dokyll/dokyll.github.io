# Scrivi il tuo blog in HTML

## Fare le pagine

Creiamo una cartella chiamata **blog**. Qui faremo le pagine del nostro blog personale.

Creiamo ora `index.html` e scriviamo:

```html
<title>Il mio blog</title>

<a href="index.html">Il mio Blog</a> - <a href="about.html">Su di me</a>

<h1>Questo è il mio blog!</h1>
<p>Qui scrivero su di me, e questo è il mio primo sito fatto con Dokyll!</p>
```

Ora anche  `about.html`:

```html
<title>Il mio blog</title>

<a href="index.html">Il mio Blog</a> - <a href="about.html">Su di me</a>

<h1>About</h1>
<p>Io sono un programmatore che in questo sito scrivo su di me!<br/>Questo sito, a proposito, l'ho fatto con Dokyll</p>
```

Guardate le pagine: funzionano, ma sono un pò bruttine.
Nella prossima pagina di documetazione aggiungeremo stile!
